把模板文件上传到template目录下解压

第一步、打开后台-系统-网站参数配置-把PC模板选择shoutu31。手机模板选择shoutu32，然后保存

如果你网站pc是www手机端是m，那么自适应手机就选择多域名，如果是www的一个域名就选择单域名，设置这个之前需要把网站域名设置好。

第二步、添加快捷菜单：打开网站后台-自定义菜单配置-把下面代码放进去-然后往下拉保存

首涂31配置,/template/shoutu31/admin/

首涂32配置,/template/shoutu32/admin/

第五步、设置网站域名：点击首涂31配置-站点设置-站点域名-配置好保存就可以了。

模板的默认账号密码是admin，登录后请立即修改账号密码，很多站长不修改账号密码导致网站被挂马
